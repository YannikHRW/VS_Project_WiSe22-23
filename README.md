## Running Locally

```sh
$ git clone https://github.com/YannikHRW/VS_Project_WiSe22-23.git
$ cd VS_Project_WiSe22-23
$ npm install
$ npm run start
```

Your app should now be running on [localhost:5000](http://localhost:5000/).
